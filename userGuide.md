# 美容室在庫管理システム ユーザーガイド

## ウェブサイト情報

**目的**: 美容室の商品在庫を管理し、発注履歴と使用履歴を記録できるシステムです。

**アクセス**: ログインが必要です。

## Powered by Manus

このウェブサイトは最新のテクノロジーで構築されています。

**フロントエンド**: React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui コンポーネントライブラリによる高速でモダンなユーザーインターフェース

**バックエンド**: Express 4 + tRPC 11による型安全なAPI通信、エンドツーエンドの型チェックで開発効率と品質を両立

**データベース**: MySQL/TiDBによる高性能なデータ管理、Drizzle ORMで安全なデータベース操作を実現

**認証**: Manus OAuthによるセキュアなユーザー認証システム

**デプロイメント**: 自動スケーリングインフラストラクチャとグローバルCDNによる高速配信

## ウェブサイトの使い方

### 商品の管理

商品一覧画面で全ての在庫商品を確認できます。在庫数が最小在庫数を下回ると、オレンジ色の背景で「在庫少」バッジが表示され、ページ上部にアラートカードが表示されます。

新しい商品を追加するには、「商品を追加」ボタンをクリックします。ダイアログが開いたら、商品名とカテゴリを選択し、現在の在庫数と最小在庫数を入力します。単位や単価、仕入先、備考も記録できます。「追加」ボタンをクリックすると商品が登録されます。

既存の商品を編集するには、商品一覧の右端にある鉛筆アイコンをクリックします。情報を更新して「更新」ボタンをクリックします。商品を削除する場合は、ゴミ箱アイコンをクリックして確認ダイアログで承認します。

### 発注履歴の記録

サイドバーの「発注履歴」をクリックして発注履歴画面を開きます。「発注を記録」ボタンをクリックし、商品を選択して発注日、数量、合計金額を入力します。発注を記録すると、自動的に在庫数が増加します。

### 使用履歴の記録

サイドバーの「使用履歴」をクリックして使用履歴画面を開きます。「使用を記録」ボタンをクリックし、商品を選択して使用日と数量を入力します。使用を記録すると、自動的に在庫数が減少します。

## ウェブサイトの管理

### 設定の変更

画面右上のManagement UIアイコンをクリックして管理パネルを開きます。「Settings」タブでウェブサイト名やロゴをカスタマイズできます。

### データベースの確認

Management UIの「Database」タブで、商品、カテゴリ、発注履歴、使用履歴のデータを直接確認・編集できます。左下の設定からデータベース接続情報を確認できます（SSL接続を有効にしてください）。

### ファイル管理

Management UIの「Code」タブでプロジェクトファイルを確認し、全ファイルをダウンロードできます。

## 次のステップ

Manus AIにいつでも話しかけて、機能の追加や変更をリクエストできます。

たとえば、商品の写真を追加したり、在庫レポートをエクスポートする機能を追加したり、カテゴリごとの在庫サマリーを表示することができます。
