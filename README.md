# 美容室在庫管理システム

美容室の商品在庫を管理し、発注履歴と使用履歴を記録できるシンプルなWebアプリケーションです。

## 主な特徴

- **シンプルな設計**: ログイン機能なし、オーナー専用
- **SQLiteデータベース**: ファイルベースで簡単にデプロイ可能
- **軽量**: サーバーレスでも動作可能

## 技術スタック

- **フロントエンド**: React 19 + TypeScript + Tailwind CSS + shadcn/ui
- **バックエンド**: Express 4 + tRPC 11
- **データベース**: SQLite (better-sqlite3)
- **ORM**: Drizzle ORM

## セットアップ

### 1. 依存関係のインストール

```bash
pnpm install
```

### 2. 環境変数の設定（オプション）

`.env`ファイルを作成（必須ではありません）:

```bash
# データベースパス（デフォルト: ./data/salon-inventory.db）
DATABASE_URL=./data/salon-inventory.db

# 本番環境の場合
NODE_ENV=production
PORT=3000
```

### 3. データベースの初期化

```bash
pnpm db:push
```

### 4. 開発サーバーの起動

```bash
pnpm dev
```

アプリケーションは `http://localhost:3000` で起動します。

## ビルドとデプロイ

### ビルド

```bash
pnpm build
```

### 本番環境で起動

```bash
pnpm start
```

## データベース

- SQLiteデータベースファイルは `data/salon-inventory.db` に保存されます（デフォルト）
- データベースファイルは `.gitignore` に含まれているため、バックアップを別途取ることを推奨します

## 機能

- 商品管理（追加・編集・削除）
- カテゴリ管理
- 在庫数の自動更新
- 低在庫アラート
- 発注履歴の記録と閲覧
- 使用履歴の記録と閲覧

## 注意事項

- **認証なし**: このアプリケーションは認証機能を持たないため、オーナー専用の内部ネットワークでの使用を推奨します
- **データバックアップ**: SQLiteファイルの定期的なバックアップを推奨します
